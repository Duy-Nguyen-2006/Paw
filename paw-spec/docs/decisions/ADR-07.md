
# ADR-07: Deployment connectors

- **Status:** Accepted
- **Date:** 2026-06-16
- **Deciders:** Tech Lead, PM

## Context
Deploy connectors (Vercel/Fly/etc.) are valuable but expand scope and risk before the core is proven.

## Decision
Deployment connectors are `[LATER]`. v1 stops at **local build + a demo/handoff report**.

## Consequences
- (+) De-risks and focuses the MVP.
- (-) Users deploy manually for now (documented in the final report).

## Revisit trigger
Core is stable (post-G4) and there is clear deploy demand.
